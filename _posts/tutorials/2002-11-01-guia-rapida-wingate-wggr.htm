Disclaimer
-------------------------------------------------------------------------------

	Esta guía está  escrita para los vagos que  pasan de leerse el
tutorial completo que escribí hace tiempo.
 
	En esta guía  no se explicará que es cada  cosa, sino que sólo
indicaré  los pasos  a seguir  para  configurar el  acceso a  internet
compartido utilizando como proxy el Wingate.


Prerrequisitos
--------------

    
	
  •  Red TCP/IP configurada
    • 
	
  •  La máquina servidor/wingate tendrá la ip 192.168.0.1 a lo largo de esta guía
    • 
	
  • Las máquinas sin modem tienen las IPs 192.168.0.X.
    • 



Los pasos para instalar el susodicho proxy son la siguientes:


1.- Instalación del Servidor WinGate
-------------------------------------


	Lo primero que hay q  hacer es bajar el fichero de instalación
de wingate.com e instalarlo en  la máquina con modem/internet, este PC
actuará como servidor.

	Una  vez  instalado,  el   servidor  ya  está  preparado  para
funcionar   con  todos  los  servicios  activos.  Es  conveniente  que
desactives  aquellos  servicios  que  no  vayas  a  utilizar  (telnet,
RealAudio...).


2.- Configuración de los clientes
----------------------------------


	El  siguiente  paso es  indicarles  a  los clientes  (Outlook,
Explorer, Netscape, CuteFTP, WinSMS, GetRight, Mozilla, ...) que están

de cada uno de estos programas  y activaremos las opciones de proxy. A
continuación expongo la configuración de los clientes más comunes

El WINGATE INTERNET CLIENT (WIC)

	A  partir de  la  versión 3.0  y  con la  licencia PRO  puedes
instalar en los PCs sin modem una pequeña utilidad que se encargará de
hacer  transparente  la  existencia  del proxy.  Esta  utulidad  viene
incluida  en  el  fichero   de  instalación  que  utilizaste  para  el
proxy-server, aunque esta vez se le indica que es un cliente.

	El WIC se  encarga de gestionar todas las  conexiones a través
del proxy  sin que  tengamos que configurar  a mano nada.   Para ello,
añadimos el programa del cual  queremos que gestione  las conexiones y
nos olvidamos de las configuraciones.
